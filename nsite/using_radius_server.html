Using RADIUS Server

Overview

RADIUS server integration offers potential improvements for TollGate authentication and payment processing.

Benefits

  • Offload authentication from router
  • Fleet management from performant machine
  • WPA Enterprise for network demarcation - less reliant on MAC addresses

E-cash Integration

  • Potential to accept e-cash strings before network access
  • Enhanced security through WPA enterprise
  • Reduced router load

Implementation Status

  • Initial testing completed
  • Needs version control implementation
  • Requires documentation for reproducibility
