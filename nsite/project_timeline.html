TollGate Project Timeline

Track the evolution of TollGate from initial concept to current development.

Initial Attempts (June 2024)

  • Dockerized OpenWRT for RPi
  • Native OpenWRT directly on RPi

Initial Success (July 2024)

  • Modified NoDogSplash on GLAR300m
  • E-cash melting to LNURL

Mid-Development (July-October 2024)

  • libsecp256k1 and libwally for nostr
  • OpenVPN support implementation
  • OpenNDS integration with local FAServer

Current Phase (October/November 2024)

  • Rust development for armhf and x86
  • TollBooth static captive portal
  • Remote FAServer for OpenNDS
  • Improving build processes and documentation

Near-Term Goals

  • Sufficient reliability for cohort participants
  • Test against less technical users
  • Support NUT 18, NIP 17, NIP 60/61

Real-World Use Cases

Barcelona (guifi.net)

  • Existing community network infrastructure
  • Currently using fiat payment rails

Isolated Areas

  • Individual resellers of internet access
  • Potential use Bitcoin rails instead

Long-Term Vision

  • Automatic meshing / market actor
  • Peer discovery via nostr and/or LoRa
  • Bottom-up integration with GSM stack
  • Support for multiple payment methods:
    • NWC
    • LNURLw
    • BOLT 12
