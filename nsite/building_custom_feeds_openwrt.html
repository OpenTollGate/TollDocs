Building Custom Feeds for OpenWRT

Overview

Custom feeds allow integration of TollGate packages into OpenWRT builds.

Build Options

Full Build

  • Complete OpenWRT toolchain
  • Takes ~1 hour on 16-core VPS
  • Most reliable but slow

Image Builder

  • Uses pre-compiled packages
  • Significantly faster
  • Good for minor changes
  • Command: opkg install ...

Feed Configuration

src-git packages https://git.openwrt.org/feed/packages.git src-git luci https://git.openwrt.org/project/luci.git src-git routing https://git.openwrt.org/feed/routing.git src-git telephony https://git.openwrt.org/feed/telephony.git

Cross-Compilation

  • Rust support being investigated
  • Architecture considerations (MIPS, ARM, x86)
  • Toolchain requirements
