Using TollBooth

Overview

TollBooth is a new captive portal implementation designed to make frontend development easier and more accessible.

Key Features

  • Runs static JavaScript on client devices
  • No compilation required
  • Easier maintenance and contributions
  • Frontend-focused architecture

Benefits

Development Speed

  • Quick iterations without compilation
  • Easy for frontend developers to contribute
  • Simplified testing process

Architecture

  • Manages IP tables directly
  • Decoupled frontend from binaries
  • Client-side processing

Implementation

  • Static frontend files
  • No PHP dependency
  • Browser-based processing
